# Agent Projects

Project workspaces keep executable plans and supporting artifacts together.

Create each project under `.agents/projects/<kebab-case-name>/README.md` with:

1. `# Project: <Name>`
2. `## Goal`
3. `## Context`
4. `## Scope`
5. `## Out Of Scope`
6. `## Checklist`
7. `## Acceptance Criteria`

Use plain checklist items without progress summaries or percentages. Keep each
item concrete and independently reviewable. Durable product documentation
belongs in its normal location; project plans should link to it rather than
copying it.
